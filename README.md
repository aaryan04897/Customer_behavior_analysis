# 📊 Data Analysis Project

## 1. Overview

This project demonstrates an end-to-end data analysis workflow using Python, SQL, and Power BI.

The objective of the project is to:

* Load and explore a dataset using Python
* Perform Exploratory Data Analysis (EDA)
* Clean and preprocess the data
* Run analytical queries using PostgreSQL
* Build an interactive Power BI dashboard
* Summarize insights in a structured report and presentation

This project showcases practical data analysis skills, data storytelling, and business insight generation.

## 2. Dataset

The dataset used in this project contains structured data suitable for analysis (e.g., sales, customers, transactions, or operations data).

It includes:

* Numerical features (e.g., revenue, quantity, price)
* Categorical features (e.g., region, product category)
* Date/time fields
* Unique identifiers

### Key Characteristics:

* Moderate size dataset
* Some missing values

* Inconsistent formatting in selected columns

The dataset was cleaned and standardized before analysis.
